##Weather App

#Overview:
The Weather App is a simple web application that provides real-time weather information using the Weather by API Ninja from RapidAPI. This project is built with a user-friendly interface using Bootstrap components such as cards and a navigation bar.

#Features:
Real-time Weather Data: Utilizing the Weather by API Ninja, the app fetches up-to-date weather information.

Bootstrap Components: The user interface is designed for clarity and ease of use, incorporating Bootstrap components like cards for data presentation and a navigation bar for smooth navigation.

Temperature Display: The app showcases the current temperature in degrees Celsius, providing users with instant weather insights.

Additional Information: Beyond temperature, the app displays humidity and wind information, offering a comprehensive overview of the current weather conditions.

#How to Use:
Clone the repository.
Open the index.html file in a web browser.
Explore the real-time weather information presented in an intuitive and visually appealing layout.
